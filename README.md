# Reassessing the Role of Progenitor Cells in Muscle Wasting in Cancer Cachexia 
This repository contains the pseudotime trajectory analysis code to reproduce the analyses in the manuscript: Reassessing the Role of Progenitor Cells in Muscle Wasting in Cancer Cachexia

1. "Integrated_output": Replicated integrated workflow output from analysis on C26 and Control Muscle cachectic mice model. Output was not used in the paper.
2. "MuSC": Processing and downstream analysis on Muscle Stem cells from C26 and Control integrated workflow.
3. "MuSC_monocle3": Pseudotrajectory analysis, using Monocle3, on Muscle Stem cells from C26 vs Control samples.
4.  R scripts to replicate results are available in this repository.
5.  Major Seurat objects or .cds objects are available by request to the author
